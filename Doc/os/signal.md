# 信号

## 简介

信号是Linux下常见的跨进程通信机制,特别是在父子进程之间应用很多,而

## 相关系统调用

有关的系统调用有3个,分别是:  
+ `sigaction`, 负责设定特定信号处理的方式  
+ `sigprocmask`, 负责建立特定进程的"信号掩码",从而过滤并忽略特定的信号  
+ `sigreturn`, 负责调用完成自定义信号处理函数后返回
